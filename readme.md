## Usage

1. Install

   ```sh
   npm install @kidonng/astro-source-file
   ```

2. Add component to your Astro layout

   ```astro
   ---
   import AstroSourceFile from '@kidonng/astro-source-file'
   ---
 
   <AstroSourceFile />
   ```

3. Profit!
